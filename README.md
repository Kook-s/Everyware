## 💻 프로젝트
그룹웨어를 주제로한 EveryWare는 사용하는 환경마다 필요한 기능과 그렇지 않은 기능을 사용자 환경에 
커스텀하여 사용자 경험을 높이기 위해서 제작되었습니다. 
사원, 관리자 친화적 중심으로 진행 되었습니다.
